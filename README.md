## 如何使用？

```
npm install --save @oipage/core.js
```

然后直接引入需要的工具对象使用即可，比如Timer：

```js
import { Time } from "@oipage/core.js"
```

更多具体的工具对象清单如下：

- [Time 时间&日期](./docs/Time.md)
- [Digits 数字](./docs/Digits.md)
